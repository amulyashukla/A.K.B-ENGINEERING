# AKB ENGINEERING Website

Premium React + Tailwind website for AKB ENGINEERING, built for water pump installation, borewell setup, pump repair, industrial water solutions, and nationwide expansion.

## Structure

- `index.html` - Main Vite entry point with SEO metadata.
- `package.json` - Project dependencies and scripts.
- `vite.config.js` - Vite configuration.
- `tailwind.config.js` - Tailwind CSS configuration.
- `postcss.config.js` - PostCSS configuration.
- `src/main.jsx` - React entry point.
- `src/App.jsx` - Routing and page layout.
- `src/index.css` - Tailwind base styles and custom theme styling.
- `src/components` - `Navbar`, `Footer`, and `WhatsAppButton` components.
- `src/pages` - `Home`, `Services`, `About`, `Gallery`, `Partners`, and `Contact` pages.

## Run locally

If you have Node.js installed, run:

```bash
cd akb-engineering
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Notes

- The project is ready for React Router-driven multi-page navigation.
- Tailwind CSS is configured for responsive, dark industrial styling.
- Floating WhatsApp button and premium hero sections are included.
